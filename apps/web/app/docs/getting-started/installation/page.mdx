# Installation

GPP requires Node.js 18 or later.

## npm (Recommended)

```bash
npm install -g gpp-cli
```

## pnpm

```bash
pnpm add -g gpp-cli
```

## yarn

```bash
yarn global add gpp-cli
```

## Verify Installation

```bash
gpp --version
```

You should see the version number printed:

```
0.9.52
```

## Using npx

If you prefer not to install globally, you can use npx:

```bash
npx gpp-cli init my-project
```

## System Requirements

- **Node.js**: 18.x or later
- **Operating Systems**: macOS, Linux, Windows (WSL recommended)
- **Optional**: Docker (for CMS platforms like Strapi)
