# Installation

## Prerequisites

Before installing GPP, ensure you have the following:

### Required

| Tool | Version | Purpose |
|------|---------|---------|
| **Node.js** | 18.x or later | Runs GPP CLI and JavaScript/TypeScript projects |
| **npm/pnpm/yarn** | Latest | Package management |
| **Git** | 2.x or later | Version control, template fetching |
| **Docker** | 24.x or later | All backend processes run in Docker (CMS, databases, APIs) |
| **Docker Compose** | 2.x or later | Multi-container orchestration |

### Optional (Language-Specific)

| Tool | Version | Purpose |
|------|---------|---------|
| **Go** | 1.22 or later | Go API templates (`api/go-http`) |
| **Python** | 3.11 or later | FastAPI templates |
| **Rust** | Latest stable | Rust Axum templates |
| **Flutter** | 3.x or later | Mobile app templates |

## Install GPP CLI

### npm (Recommended)

```bash
npm install -g gpp-cli
```

### pnpm

```bash
pnpm add -g gpp-cli
```

### yarn

```bash
yarn global add gpp-cli
```

## Verify Installation

```bash
gpp --version
```

You should see the version number printed:

```
0.9.98
```

## Using npx

If you prefer not to install globally, you can use npx:

```bash
npx gpp-cli init my-project
```

## Verify Prerequisites

Check that required tools are installed:

```bash
# Node.js
node --version

# Git
git --version

# Docker (recommended)
docker --version
docker compose version
```
